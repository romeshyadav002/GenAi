# GenAI Udemy Learning

This repository contains Python projects and learning materials from the GenAI Udemy course.

## Setup Instructions

### Prerequisites
- Python 3.12 or higher
- Conda (Miniconda or Anaconda)
- Git

### Steps

1. **Installed Python** - Set up Python 3.12
2. **Installed Miniconda** - Lightweight conda distribution for environment management
3. **Created Virtual Environment** - Isolated Python environment for dependencies

### Creating the Environment

```powershell
conda create -p venv python=3.12
conda activate .\venv
```

### Installing Dependencies

```powershell
pip install -r requirements.txt
```

## Project Structure
